# JSON Power Tools

A sleek, high-performance JSON utility suite built with React and Tailwind CSS. Featuring a modern "liquid glass" aesthetic and a powerful set of developer tools.

## Features

- **JSON Formatter**: Beautify messy JSON strings with proper indentation.
- **JSON Validator**: Instantly check if your JSON is syntactically correct.
- **JSON Converter**: Convert JSON data to CSV format for easy spreadsheet usage.
- **JSON Explorer**: Interactive tree view to navigate complex JSON structures.

## Design

- **Main Color**: ORANGEX (#D97757)
- **Secondary Colors**: GREYX (#808080) and WHITEX (#FFFFFF)
- **Aesthetic**: Liquid Glass effect with backdrop blurs and subtle borders.

## Usage

1. Paste your JSON data into the input area.
2. Select a tool from the navigation bar.
3. Click the action button to process your data.
4. Copy the result or explore the structure.

## Deployment

This is a static website that can be hosted on GitHub Pages or any static hosting service.
